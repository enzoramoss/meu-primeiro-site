Sobre o Repositório Inicial

Este repositório contém um projeto pessoal, desenvolvido durante o segundo bimestre. Trata-se de um Simulador de Batalha Pokémon.

Você pode acessar o repositório do projeto no GitHub através deste link.

Principais Funcionalidades

  • Simulação de batalhas entre Pokémons
  • Capacidade de personalização da experiência

Sobre o Projeto

As principais funcionalidades do Simulador de Batalha Pokémon incluem:

  • Seleção de Pokémon: Permite aos usuários escolher seus Pokémons para batalhar.
  • Sistema de Batalha: Simula as batalhas entre os Pokémons escolhidos, considerando suas habilidades e estatísticas.
  • Movimentos e Habilidades: Cada Pokémon pode usar movimentos específicos, impactando a batalha de maneiras diferentes.
  • Interface Interativa: Proporciona uma interface amigável para que os usuários possam acompanhar a batalha em tempo real.
  • Resultados das Batalhas: Exibe os resultados das batalhas após o término, informando quem venceu e as condições da luta.

Essas funcionalidades proporcionam uma experiência divertida para os fãs de Pokémon, permitindo que eles simulem batalhas de forma bem simples
